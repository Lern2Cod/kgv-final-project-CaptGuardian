- Tombol 's' untuk memutar objek kearah sumbu y negatif
- Tombol 'a' untuk memutar objek kearah sumbu x negatif

- Tombol 'arrow up' untuk menggeser kamera/sudut pandang kearah sumbu y positif
- Tombol 'arrow down' untuk menggeser kamera/sudut pandang kearah sumbu y negatif

- Tombol 'arrow 'right' untuk menggeser kamera/sudut pandang kearah sumbu x positif
- Tombol 'page up' untuk menggeser kamera/sudut pandang kearah sumbu z negatif
- Tombol 'page down' untuk menggeser kamera/sudut pandang kearah sumbu z positif
